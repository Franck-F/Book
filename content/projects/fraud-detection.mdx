# Fraud Detection System

Ingénierie de données et Machine Learning appliqués à la cybersécurité financière.

## Architecture
- Pipeline de données temps réel avec Kafka.
- Modèle de détection d'anomalies (Autoencoders & LSTM).
- Déploiement conteneurisé pour une scalabilité horizontale.
