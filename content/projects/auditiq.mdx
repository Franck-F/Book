# AuditIQ - Plateforme d'Audit Intelligente

## Vue d'ensemble

AuditIQ est une application full-stack moderne conçue pour simplifier et automatiser le processus d'audit. Utilisant l'intelligence artificielle via l'API Gemini, la plateforme offre des analyses automatiques et des recommandations personnalisées.

## Fonctionnalités principales

- **Gestion d'audits** : Création, suivi et archivage d'audits
- **Analyse IA** : Génération automatique de rapports avec Gemini
- **Dashboard interactif** : Visualisation des données en temps réel
- **Authentification** : Système sécurisé avec Supabase Auth
- **Responsive** : Interface adaptée à tous les écrans

## Technologies utilisées

- **Frontend** : Next.js 14, React, TypeScript
- **Backend** : Supabase (PostgreSQL, Auth, Storage)
- **IA** : Google Gemini API
- **Styling** : Tailwind CSS, shadcn/ui
- **Déploiement** : Netlify

## Défis techniques

Le principal défi était d'intégrer l'API Gemini de manière efficace tout en maintenant des performances optimales. J'ai implémenté un système de cache et de traitement asynchrone pour gérer les requêtes IA.

## Résultats

- ✅ Temps de génération de rapport réduit de 80%
- ✅ Interface utilisateur intuitive et moderne
- ✅ 100% responsive sur tous les appareils
- ✅ Déploiement automatisé avec CI/CD
