# Prédiction des Admissions Hospitalières

## Objectif

Développer un modèle de machine learning pour prédire les admissions hospitalières et optimiser la gestion des ressources.

## Approche

Comparaison de deux algorithmes de gradient boosting :
- LightGBM
- XGBoost

## Résultats

LightGBM sélectionné comme modèle champion avec :
- MAE : 12.5
- R² : 0.89
- Temps d'entraînement : 3x plus rapide que XGBoost
