# DeepLearn Amazon Sentiment Analysis

Ce projet propose une architecture de Deep Learning (**LSTM**) pour classifier les sentiments des avis Amazon, complétée par une analyse exploratoire (**EDA**) interactive.

## Points Clés
- **EDA Interactive** : Visualisations dynamiques avec **Plotly**.
- **Deep Learning** : Architecture de neurones récurrents (**LSTM**) pour la classification.
- **Performance** : Gestion optimisée des packages avec **uv**.

## Résultats
Le dashboard permet de visualiser en temps réel la précision du modèle et l'impact des mots-clés sur la polarité des sentiments détectés.
