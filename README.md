A high-performance portfolio and research journal designed for AI Engineers and Data Scientists. Built with a focus on technical depth, premium aesthetics, and high-fidelity content rendering.

## Core Features

- **Advanced MDX Engine**: Custom rendering for technical content, including syntax highlighting, interactive code blocks, and markdown tables.
- **Topical Categorization**: Content organized by thematic expertise (LLM Architectures, Deep Learning, Data Engineering).
- **Social Interactivity**: Native share and bookmarking features integrated with local storage and the Web Share API.
- **Glassmorphism Design System**: A premium, dark-themed UI built with Tailwind CSS v4 and Framer Motion.
- **SEO Optimized**: Dynamic metadata generation for all technical articles and projects.

## Technical Architecture

| Component | Technology |
| :--- | :--- |
| **Framework** | Next.js 16 (App Router) |
| **Language** | TypeScript |
| **Styles** | Tailwind CSS v4 (Rust-based engine) |
| **Animations** | Framer Motion |
| **Icons** | Lucide React |
| **Content Management** | Local MDX with gray-matter parsing |

## Content Management Guidelines

### Research Journal (Blog)

Articles should be placed in `content/blog/` with standard frontmatter including title, date, category, and readingTime.
